# EarlyAD-biomarker-temporal
Early-stage Alzheimer’s biomarker discovery (Temporal cortex)

This repository provides a reproducible transcriptomic pipeline for identifying robust early-stage Alzheimer’s disease (AD) biomarkers from temporal cortex datasets.

The workflow integrates multiple bulk transcriptomic datasets and single-nucleus RNA sequencing datasets to identify cross-platform consensus biomarkers associated with preclinical AD progression.

The analysis strategy combines:

Differential expression analysis

PCA-based gene expansion

Network filtering using WGCNA

Elastic Net feature selection

Cross-platform validation using snRNA-seq

The goal is to identify robust early-stage AD biomarker genes consistently detected across independent datasets and platforms.
